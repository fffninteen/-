- ==**Поставляемый результат содержит некоторый набор артефактов**==
- Поставляемый результат обязательно содержит более одного артефакта
- Поставляемый результат содержит ровно один артефакт
- Поставляемый результат является артефактом
- Поставляемый результат описывает артефакт
